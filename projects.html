EMT Route Learning

Technologies: HTML, CSS, JavaScript, Google Routes API

Designed a game that allows users to test their navigational knowledge by quizzing them on how one can drive between two locations. Users enter a list of destinations, and a singular starting location, and the game generates fill-in-the-blank sentences, prompting the user to enter street names they must take to get to the end place.
