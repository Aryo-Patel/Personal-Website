War At Sea

Technologies: MongoDB, Express, React, Node.js, Cloudinary, CPanel

Programmed a game that allows users to build a WW2 naval fleet. Each ship is assigned a point value, and the user decide the max points the Allies and the Axis can have. Users can meticulously choose each ship in their squadron, or allow the computer to randomly generate a squadron for them.
